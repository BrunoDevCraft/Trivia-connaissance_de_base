Jeu du Trivia

	Bienvenue dans le Jeu du Trivia, un jeu de quiz simple créé en utilisant Python et Pygame.

Comment Jouer:

	=> Lancez le jeu en exécutant le script Trivia_0.1.py.
	=> À l'écran de présentation, cliquez sur le bouton "START" pour démarrer.
	=> Dans l'écran de sélection de thème, choisissez un thème en cliquant sur le numéro correspondant.
	=> Répondez aux questions en appuyant sur les touches 1, 2 ou 3 pour les options à choix multiples.
	=> Vous pouvez également cliquer sur les options avec votre souris.
	=> Continuez à répondre aux questions jusqu'à épuisement des vies ou jusqu'à la fin du quiz.
	=> Après le jeu, le score final sera affiché.

Structure des Fichiers:
	=> Trivia_0.1.py: Script principal pour exécuter le Jeu du Trivia.
	=> assets/Themes/: Répertoire contenant les fichiers de questions spécifiques à chaque thème.

Prérequis:

	=> Python 3.x
	=> Bibliothèque Pygame

Comment Installer Pygame:

	=> Pour installer Pygame, ouvrez un terminal ou une invite de commande et exécutez :

			=> pip install pygame
